# ScoreFlow Coach

`ScoreFlow-Coach` is the dedicated home for the coach-facing ScoreFlow application.

## Boundary

The existing ScoreFlow scoreboard remains in the repository root and is intentionally unchanged by this foundation. Coach-specific pages, styles, scripts, PWA behavior, Firebase integration, and future modules should stay inside this folder unless a later pull request explicitly defines a shared package or shared data contract.

## Foundation Navigation

- Home
- Teams
- Players
- Rosters
- Live Match
- Rotations
- Reports
- Settings

## Current State

This folder currently contains a responsive dashboard shell and placeholder views. It does not yet connect to Firebase or modify scoreboard data.
