# ScoreFlow Product Vision

ScoreFlow is the operating system for volleyball programs.

The existing ScoreFlow scoreboard remains the live scoring and broadcast engine. ScoreFlow Coach becomes the daily workspace for coaches, assistants, players, and parents.

## Core direction

- Volleyball first
- Coach focused
- Mobile and iPad primary
- Offline first
- Integrated with the live ScoreFlow scoreboard
- Clean, bold, premium, and broadcast ready
- Powerful enough for serious programs without feeling intimidating

## Product family

- **ScoreFlow Scoreboard** — live scoring, viewer links, fan engagement, ticker graphics, results, and shareable graphics
- **ScoreFlow Coach** — teams, seasons, rosters, schedules, practices, rotations, live stats, reports, and player development
- **ScoreFlow Team Portal** — read-only player and parent access to schedules, results, selected stats, player cards, and coach-shared content

## Core workflow

Organization → Team → Season → Roster → Schedule → Live Match → Match Report → Season and Player Statistics

The coach should not need to enter the same information more than once.
