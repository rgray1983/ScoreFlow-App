# PR-009 — Fullscreen Live Match + Two-Touch Stats

## Purpose

Create ScoreFlow Coach's first usable courtside Match Mode: a fullscreen, offline-first interface where the coach can control the official score and record player statistics without leaving the court view.

## Included

- Fullscreen-style Match Mode that hides normal app navigation
- Persistent Home control with Resume, Save & Exit, and Reset choices
- Browser fullscreen control where supported
- Active match context from the current team, season, and scheduled opponent
- Independent Home and Away score controls
- Serve indicator, set and rotation display
- Court-position player buttons
- Two-touch player → stat entry
- Kill, error, ace, dig, block, assist, and pass-rating events
- Visual rotation of court players
- Bench list and substitution controls
- Timeout, undo-event, and timeline controls
- Locally persisted score, court, bench, rotation, and event timeline
- Responsive iPad, phone landscape, and phone portrait layouts
- Six additional prototype players, for nine total test players
- Safe seed migration that adds missing prototype players without deleting user-created data

## Core rule

Score and statistics are independent. A statistical event never awards a point automatically. The scorer records the official point separately after the whistle.

## Deferred
