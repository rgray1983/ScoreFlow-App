# PR-008 — Practice Planning Foundation

## Purpose

Turn scheduled Practice events into structured, reusable coaching plans without requiring an internet connection.

## Included

- Working `/practice` route and primary navigation item
- Active Organization → Team → Season context
- Scheduled-practice selector sourced from Schedule
- Timed practice timeline with total duration
- Custom practice blocks with category, duration, equipment, and notes
- Quick drill library for common volleyball activities
- Block reordering and removal
- Primary focus, coach notes, and completion status
- Local persistence per scheduled practice
- Shared ScoreFlow dropdown and slide-over components
- Phone portrait, phone landscape, and iPad responsive layouts
- Coach Hub quick action and next-practice handoff

## Deferred

- Creating schedule events from Practice
- Full reusable templates
- Custom drill-library management
- Drag-and-drop timeline ordering
- Attendance and player groups
- Live practice timer
- Sharing and assistant collaboration
- Firebase, authentication, and synchronization
