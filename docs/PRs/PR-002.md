# PR-002 — Create ScoreFlow Coach App Foundation

## Status

Ready for local testing and review.

## Branch

`agent/pr-002-scoreflow-coach-foundation`

## Purpose

Create a separate, scalable `ScoreFlow-Coach` application for coach-facing features without changing the existing ScoreFlow scoreboard app in the repository root.

## Scope

- Create the dedicated `ScoreFlow-Coach` folder.
- Build the Coach app with React, TypeScript, and Vite.
- Add typed routes for Home, Teams, Players, Rosters, Live Match, Rotations, Reports, and Settings.
- Add a responsive desktop sidebar and mobile navigation drawer.
- Add generated PWA support through `vite-plugin-pwa`.
- Add TypeScript type checking and production build scripts.
- Document the boundary between the existing scoreboard and the Coach app.
- Do not connect Firebase yet.
- Do not modify existing scoreboard files.

## Implementation Notes

The Coach app is now an independent Vite application using React and TypeScript. React Router provides real client-side routes, and the PWA plugin generates the web manifest and service worker during production builds.

The original temporary static `app.js`, `style.css`, `manifest.json`, and `service-worker.js` files were removed rather than left as conflicting legacy code.

No Firebase configuration, authentication, team data, player data, match data, or scoreboard integration is included. The existing ScoreFlow scoreboard files in the repository root remain intentionally untouched.
