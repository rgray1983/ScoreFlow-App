# PR-010 — Match Setup, Rules, and Completion

## Goal

Turn the PR-009 Live Match prototype into a complete volleyball match lifecycle that can be configured, scored, corrected, completed, saved, and reopened.

## Core workflow

1. Select a scheduled match or create a Quick Match.
2. Confirm home/away, opponent, format, scoring targets, initial serve, starting six, Libero, and initial rotation.
3. Enter the existing fullscreen Live Match court.
4. Complete sets using volleyball win conditions.
5. Rotate, substitute, use Libero replacements, record stats, and undo mistakes safely.
6. Complete and permanently save the match.
7. Reopen the saved match or continue to its summary.

## Included

- Persistent Match and MatchSet records in the workspace data model
- Safe migration for existing locally saved workspaces
- Scheduled-match selector and Quick Match setup
- Best-of-three and best-of-five formats
- Configurable regular and deciding set targets with win-by-two
- Starting six, Libero, initial rotation, and initial serve confirmation
- Set-complete confirmation and next-set transition
- Match-complete summary and scheduled-event completion
- Full undo snapshots for score, serve, rotation, timeout, substitution, stat, and set transitions
- Serve-aware rotation behavior with manual correction
- Drag-and-drop substitutions with accessible fallback controls
- Libero replacement workflow that remains separate from normal substitutions
- Permanent saved match records for future Reports and season analytics

## Rules

- Score and statistics remain independent.
- The Serve control remains manually tappable even when serve changes automatically after a point.
- Automatic rotations must remain visible and undoable.
- The app should guide volleyball rules without trapping a coach when a real-world correction is required.
- Existing PR-009 Live Match visual behavior and responsive layouts must remain intact unless a lifecycle feature requires a focused change.

## Deferred

- Attack and serve zones
- Linked rally sequences
- Opponent-player statistics
- Full reports and season leaderboards
- Cloud sync and multi-device collaboration
- Tryouts

## Definition of done

A coach can open a scheduled or quick match, configure it, score and stat every set, rotate and substitute players, use Libero replacements, undo mistakes, finish the match, save it permanently, mark its schedule event complete, and reopen the result.
