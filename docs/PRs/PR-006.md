# PR-006 — Schedule Foundation

## Status

Ready for local testing and review.

## Branch

`agent/pr-006-schedule-foundation`

## Purpose

Give coaches a working local-first schedule for matches, practices, tournaments, scrimmages, and team events. Schedule records belong to the active team and season and are structured to flow into future pre-match, Live Match, portal, attendance, and reporting workflows.

## Scope

- Add typed schedule event, status, and location models.
- Add schedule event persistence and safe migration to the existing workspace store.
- Add representative match and practice seed records.
- Add Schedule to primary phone and iPad navigation.
- Reuse the Organization → Team → Season Context Bar.
- Add Agenda, Upcoming, and Month views.
- Add event creation and editing through the shared slide-over editor.
- Reuse ScoreFlow custom dropdowns and date picker.
- Use 15-minute custom time selectors rather than browser-native time controls.
- Support event type, title, date, start/arrival/end time, opponent, home/away/neutral, venue, address, court, travel notes, event notes, and status.
- Update the Coach Hub to show the next scheduled event and upcoming-event totals.
- Preserve the existing root scoreboard.
