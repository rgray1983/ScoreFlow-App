# Practice Planning

## Purpose

Turn match observations and season goals into organized, reusable practices.

## Implemented foundation

- Open scheduled Practice events inside the active organization, team, and season
- Build a timed practice timeline
- Add custom blocks with category, duration, notes, and equipment
- Add common blocks from a reusable quick-drill library
- Reorder and remove timeline blocks
- Track total planned minutes
- Save primary focus, coaching notes, and completion status locally
- Use the workspace without an internet connection
- Responsive phone and iPad layouts

## Planned capabilities

- Create or duplicate practices directly from the Practice workspace
- Save full plans as reusable templates
- Custom organization drill libraries
- Drag-and-drop timeline ordering
- Attendance and player groups
- Share selected plans with assistants or players
- Post-practice review and completion notes

## Future connections

- Suggested focus areas from recent match statistics
- Rotation-specific drills
- Player development goals
- Recurring practice schedules
- Live timers during practice

Practice planning should remain useful without requiring advanced statistics or an internet connection.
