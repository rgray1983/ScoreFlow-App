# Schedule

## Purpose

Give coaches one place to plan matches, practices, tournaments, and team events. Scheduled information should flow into live matches, portals, reminders, reports, and attendance without duplicate entry.

## Event types

- Match
- Practice
- Tournament
- Scrimmage
- Team event

## Core fields

- Team and season
- Event type and title
- Date, start time, arrival time, and optional end time
- Opponent
- Home, away, or neutral designation
- Venue, address, court assignment, and travel notes
- Event notes
- Recurrence for practices and team events
- Status: draft, scheduled, changed, cancelled, completed

## Planned views

- Agenda
- Month calendar
- Upcoming list
- Team-specific schedule

## Match workflow

Scheduled Match → Pre-match preparation → Starting lineup → Live Match → Completed Match → Match Report

Opponent, venue, date, team, season, and roster context must carry forward automatically.

## Later capabilities

- Multi-day tournaments and multiple matches per day
- Pool and bracket organization
- Player availability
- Attendance
- Transportation coordination
- Parent/player notifications
- Google, Apple, and Outlook calendar sync
- CSV and common schedule imports

Schedule is a first-class module planned after Teams, Seasons, Players, and Rosters and before the complete Live Match implementation.
