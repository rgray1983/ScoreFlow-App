# Reports

## Report types

- Match report
- Team season report
- Player season report
- Rotation report
- Practice and attendance report
- Opponent comparison

## Initial content

- Final score and set results
- Match timeline
- Key player statistics
- Team totals
- Side-out and serving-run summaries
- Rotation performance
- Coach notes and corrections

## Outputs

- In-app responsive report
- Shareable result card
- PDF export
- CSV or spreadsheet export where appropriate
- Coach-approved portal summary

## Principles

Reports read from the event and entity model rather than maintaining a disconnected reporting database. Corrections to a match must flow through to reports with an audit trail.
