# Live Match Hub

## Purpose

Bring scoreboard control, two-touch stat entry, current rotation, substitutions, timeouts, notes, and timeline into one reliable coach workflow.

## iPad direction

- Compact live scoreboard controls
- Active roster/player cards
- Large stat workflow
- Match timeline and undo
- Current rotation and substitution access

## Phone direction

- Persistent compact score strip
- Fast tabs or swipeable panels
- Bottom navigation for roster, stats, rotation, timeline, and share
- Undo always within easy reach

## Data principles

- Start from a scheduled match when one exists
- Save every important action locally first
- Record events with unique IDs, timestamps, and sequence order
- Update scoreboard, totals, and timeline from the event stream
- Preserve corrections and undo history
- Continue operating without internet

## Initial stat workflow

Tap player → tap result. Begin with a simple configurable set of stats before exposing advanced contact tracking.
