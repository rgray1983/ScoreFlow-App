# Statistics

## Purpose

Provide serious volleyball information through a fast, configurable workflow that does not force every coach to track every contact.

## Initial event results

- Kill
- Attack error
- Ace
- Serve in
- Serve error
- Block
- Dig
- Assist
- Pass ratings
- Free ball
- Substitution
- Timeout
- Coach note

## Principles

- Simple mode comes first.
- Advanced detail is optional.
- Match events are the source of truth; totals are derived or safely cached.
- Undo and correction preserve event history.
- Stats connect to players, teams, seasons, matches, sets, and rotations.
- All live entries save locally before cloud synchronization.

## Future analysis

- Side-out percentage
- Serving runs
- Rotation efficiency
- Passing and attack trends
- Momentum changes
- Heat maps and shot charts
- Opponent scouting
- Player development trends
