# Seasons

## Purpose

Separate rosters, schedules, statistics, and reports into clear competitive periods without duplicating reusable player profiles.

## Planned capabilities

- Create named seasons and date ranges
- Set active, upcoming, completed, and archived status
- Copy selected roster members or settings from a previous season
- Associate schedules, matches, practices, rotations, and reports
- Preserve historical statistics after a new season begins

## Future connections

The active season controls the default roster, Coach Hub schedule, reports, and live match context. A team may have multiple stored seasons, but only explicitly selected seasons should drive current dashboard data.
